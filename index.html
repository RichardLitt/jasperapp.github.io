Overview

Jasper is a tool for reading issues and pull requests in GitHub efficiently. You will be able to read issues by condition of author, repository, organization and etc.

Your Issues

By default, Jasper automatically shows issues - you are involved in, your team is mentioned in, and of repository you watch. You'll never miss issues.

Stream

Jasper supports the streaming - show issues in a particular repository, of a particular user, with a particular word and etc. You can read active issues in GitHub.

Filter

Jasper supports the powerful filtering - select unread issues, contains particular labeled issues, no assignee issues and etc. You can easily and quickly find what you want to check out now.

Sort

Jasper supports the powerful sorting - rearrange the order of issues by update date, read date, due date and etc. You can read issues according to interest.
